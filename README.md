# Sales-Management-System--PYTHON
This project was developed as a first-year Python learning project.

Sales Pro - Sales Management System
About the Project: 

This is a Sales Management System developed as a second-semester Python project. I built this project to understand how a real-world desktop application works by combining Python, Tkinter, and MongoDB.
The application allows users to manage sales records, perform CRUD operations, analyze sales data, generate reports, and visualize business performance through an easy-to-use graphical interface.
This project also helped me learn modular programming, database connectivity, GUI development, and role-based authentication.
